== Description ==

Adds Georgian lari (GEL) symbol to your WooCommerce shop.

If you would like to accept credit and debit card payments in your WooCommerce shop via TBC bank gateway: [WooCommerce Tbc Payment Gateway](https://wordpress.org/plugins/woo-tbc-payment-gateway/)

[font](https://github.com/Stichoza/font-larisome) by **Stichoza**

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/woo-currency-gel` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the WooCommerce->Settings screen, in currency options select "Georgian Lari (GEL)"
4. Done!
